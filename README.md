# influxdb-to-hdf5
Script to gather times-series from an InfluxDB server and save the datasets in HDF5 files

**Requirements:**  
* Python>=2.7.11
* numpy>=1.12.0
* h5py>=2.7.0
* influxdb>=4.0.0
* jsonschema>=2.6.0

**Usage:** 
<pre>bridge.py -c conf/example.json</pre>

See conf/example.json for the configuration's description.
